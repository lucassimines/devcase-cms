# Devcase Admin

Nuxt 4 admin panel for [Devcase CMS](../README.md).

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs at `http://localhost:3000`.

## Production

```bash
npm run build
npm run preview
```

## Quality

```bash
npm run lint
npm run typecheck
npm test
```
